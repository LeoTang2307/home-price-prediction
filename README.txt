*********************** MAIN PURPOSE ***********************
I implemented a basics machine learning model called Gradient Boosting Regressor to predict the house price in King Country of the USA at the time between May 2014 and May 2015.

*********************** RUNNING ***********************
Open the projject folder in vscode or navigate to it in your command line on your computer (ex: project A would have the folder named "project_A")
Run "docker-compose up" in the terminal of vscode or command line on your computer
